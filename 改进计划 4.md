### **核心目标**
实现**个人历史排行榜**功能，让用户与自己的历史表现对比，增强成就感，提升用户留存率 10-15%。

### **功能范围**
1. **个人历史对比** - 本周 vs 上周 vs 历史最佳
2. **成就系统增强** - 新增个人里程碑成就
3. **分享功能** - 生成可分享的环保证书图片

### **设计原则**
- ✅ **隐私优先** - 100% 本地数据，无需网络
- ✅ **个人成长** - 只与自己比较，无社交压力
- ✅ **正向激励** - 强调进步，而非绝对数值
- ✅ **简洁美观** - 符合当前软件 UI 风格
- ✅ **代码复用** - 优先复用现有模块

/// 个人表现数据
struct PersonalPerformance: Identifiable, Codable {
    let id: String
    let period: PerformancePeriod  // .weekly, .monthly, .yearly
    let totalCO2: Double
    let activityCount: Int
    let averagePerDay: Double
    let bestDay: Double
    let startDate: Date
    let endDate: Date
    
    /// 格式化 CO2 减排量
    var formattedCO2: String {
        String(format: "%.1f lbs", totalCO2)
    }
    
    /// 格式化天数
    var daysDescription: String {
        let days = Calendar.current.dateComponents([.day], from: startDate, to: endDate).day ?? 0
        return "\(days) days"
    }
}

/// 表现对比结果
struct PerformanceComparison {
    let current: PersonalPerformance
    let previous: PersonalPerformance?
    let best: PersonalPerformance?
    
    /// 相比上周的变化百分比
    var changePercent: Double {
        guard let previous = previous, previous.totalCO2 > 0 else { return 0 }
        return ((current.totalCO2 - previous.totalCO2) / previous.totalCO2) * 100
    }
    
    /// 是否进步（减排更多）
    var isImproved: Bool {
        changePercent < 0  // 负数表示减排更多
    }
    
    /// 变化描述
    var changeDescription: String {
        if changePercent == 0 { return "No change" }
        let sign = changePercent < 0 ? "↓" : "↑"
        return "\(sign) \(String(format: "%.1f", abs(changePercent)))%"
    }
}

/// 时间周期
enum PerformancePeriod: String, Codable {
    case weekly = "Week"
    case monthly = "Month"
    case yearly = "Year"
}

/// 百分位排名结果
struct PercentileRanking {
    let percentile: Double  // 0-100
    let rank: String
    let message: String
    let icon: String
    
    static func from(percentile: Double) -> PercentileRanking {
        switch percentile {
        case 90...:
            return PercentileRanking(
                percentile: percentile,
                rank: "Top 10%",
                message: "🏆 Best performance ever! You're beating 90% of your history!",
                icon: "trophy.fill"
            )
        case 70..<90:
            return PercentileRanking(
                percentile: percentile,
                rank: "Great",
                message: "🌟 Great job! Better than most of your history!",
                icon: "star.fill"
            )
        case 50..<70:
            return PercentileRanking(
                percentile: percentile,
                rank: "Good",
                message: "👍 Good effort! Above average!",
                icon: "checkmark.circle.fill"
            )
        default:
            return PercentileRanking(
                percentile: percentile,
                rank: "Keep Going",
                message: "💪 Keep going! Every step counts!",
                icon: "leaf.fill"
            )
        }
    }
}

import Foundation

@MainActor
final class PersonalRankingService: ObservableObject {
    @Published var currentWeekPerformance: PersonalPerformance?
    @Published var performanceComparison: PerformanceComparison?
    @Published var percentileRanking: PercentileRanking?
    
    private let repository: ActivityRepositoryProtocol
    
    init(repository: ActivityRepositoryProtocol = CoreDataActivityRepository()) {
        self.repository = repository
    }
    
    // MARK: - Public Methods
    
    /// 加载本周表现数据
    func loadCurrentWeekPerformance() async throws {
        let calendar = Calendar.current
        let now = Date()
        
        // 获取本周开始时间
        let startOfWeek = calendar.startOfWeek(for: now)
        
        // 获取本周活动数据
        let activities = try await repository.fetchActivities(
            from: startOfWeek,
            to: now
        )
        
        // 计算本周总减排量
        let totalCO2 = activities.reduce(0.0) { $0 + $1.co2Emission }
        
        // 计算每天的平均值
        let daysSoFar = max(1, calendar.dateComponents([.day], from: startOfWeek, to: now).day ?? 1)
        let averagePerDay = totalCO2 / Double(daysSoFar)
        
        // 找出最佳单日
        let dailyTotals = Dictionary(grouping: activities) { activity in
            calendar.startOfDay(for: activity.date)
        }.mapValues { activities in
            activities.reduce(0.0) { $0 + $1.co2Emission }
        }
        let bestDay = dailyTotals.max(by: { $0.value < $1.value })?.value ?? 0
        
        currentWeekPerformance = PersonalPerformance(
            id: UUID().uuidString,
            period: .weekly,
            totalCO2: totalCO2,
            activityCount: activities.count,
            averagePerDay: averagePerDay,
            bestDay: bestDay,
            startDate: startOfWeek,
            endDate: now
        )
    }
    
    /// 加载历史对比数据
    func loadPerformanceComparison() async throws {
        let calendar = Calendar.current
        let now = Date()
        
        // 本周数据
        let startOfWeek = calendar.startOfWeek(for: now)
        let currentWeekActivities = try await repository.fetchActivities(
            from: startOfWeek,
            to: now
        )
        let currentWeekCO2 = currentWeekActivities.reduce(0.0) { $0 + $1.co2Emission }
        
        // 上周数据
        let lastWeekStart = calendar.date(byAdding: .weekOfYear, value: -1, to: startOfWeek)!
        let lastWeekActivities = try await repository.fetchActivities(
            from: lastWeekStart,
            to: startOfWeek
        )
        let lastWeekCO2 = lastWeekActivities.reduce(0.0) { $0 + $1.co2Emission }
        
        // 历史最佳周数据（过去 12 周）
        let twelveWeeksAgo = calendar.date(byAdding: .weekOfYear, value: -12, to: startOfWeek)!
        let allActivities = try await repository.fetchActivities(
            from: twelveWeeksAgo,
            to: now
        )
        
        // 按周分组
        let weeklyTotals = Dictionary(grouping: allActivities) { activity in
            calendar.dateComponents([.yearForWeekOfYear, .weekOfYear], from: activity.date)
        }.mapValues { activities in
            activities.reduce(0.0) { $0 + $1.co2Emission }
        }
        
        let bestWeekCO2 = weeklyTotals.values.max() ?? 0
        
        // 构建对比结果
        let current = PersonalPerformance(
            id: UUID().uuidString,
            period: .weekly,
            totalCO2: currentWeekCO2,
            activityCount: currentWeekActivities.count,
            averagePerDay: currentWeekCO2 / 7,
            bestDay: 0,
            startDate: startOfWeek,
            endDate: now
        )
        
        let previous = lastWeekCO2 > 0 ? PersonalPerformance(
            id: UUID().uuidString,
            period: .weekly,
            totalCO2: lastWeekCO2,
            activityCount: lastWeekActivities.count,
            averagePerDay: lastWeekCO2 / 7,
            bestDay: 0,
            startDate: lastWeekStart,
            endDate: startOfWeek
        ) : nil
        
        let best = bestWeekCO2 > 0 ? PersonalPerformance(
            id: UUID().uuidString,
            period: .weekly,
            totalCO2: bestWeekCO2,
            activityCount: 0,
            averagePerDay: bestWeekCO2 / 7,
            bestDay: bestWeekCO2,
            startDate: twelveWeeksAgo,
            endDate: now
        ) : nil
        
        performanceComparison = PerformanceComparison(
            current: current,
            previous: previous,
            best: best
        )
    }
    
    /// 计算百分位排名
    func calculatePercentileRanking() async throws {
        guard let current = currentWeekPerformance else { return }
        
        // 获取过去 12 周的数据
        let calendar = Calendar.current
        let now = Date()
        let twelveWeeksAgo = calendar.date(byAdding: .weekOfYear, value: -12, to: now)!
        
        let allActivities = try await repository.fetchActivities(
            from: twelveWeeksAgo,
            to: now
        )
        
        // 按周分组计算每周总量
        let weeklyTotals = Dictionary(grouping: allActivities) { activity in
            calendar.dateComponents([.yearForWeekOfYear, .weekOfYear], from: activity.date)
        }.mapValues { activities in
            activities.reduce(0.0) { $0 + $1.co2Emission }
        }
        
        let allWeeks = Array(weeklyTotals.values)
        
        // 计算有多少周的减排量少于本周
        let weeksWithLessCO2 = allWeeks.filter { $0 < current.totalCO2 }.count
        
        // 计算百分位
        let percentile = allWeeks.isEmpty ? 0 : 
            Double(weeksWithLessCO2) / Double(allWeeks.count) * 100
        
        percentileRanking = PercentileRanking.from(percentile: percentile)
    }
    
    /// 获取个性化鼓励文案
    func getEncouragementMessage() -> String {
        guard let ranking = percentileRanking else {
            return "Start logging activities to see your progress!"
        }
        return ranking.message
    }
}

extension AchievementService {
    // MARK: - Personal Ranking Integration
    
    /// 检查个人排名相关成就
    func checkPersonalRankingAchievements(percentile: Double) async {
        // 检查是否达到 Top 10%
        if percentile >= 90 {
            await checkAndUnlockAchievement(id: "top_10_percent")
        }
    }
    
    /// 检查是否打破个人最佳
    func checkPersonalBestAchievement(currentCO2: Double) async {
        do {
            // 获取历史最佳周
            let historicalBest = try await getHistoricalBestWeek()
            
            if currentCO2 > historicalBest {
                await checkAndUnlockAchievement(id: "improvement_king")
            }
        } catch {
            print("Error checking personal best: \(error)")
        }
    }
    
    /// 检查完美周成就
    func checkPerfectWeekAchievement() async {
        do {
            let calendar = Calendar.current
            let now = Date()
            let startOfWeek = calendar.startOfWeek(for: now)
            
            let activities = try await repository.fetchActivities(
                from: startOfWeek,
                to: now
            )
            
            // 按天分组
            let daysWithActivities = Set(activities.map { 
                calendar.startOfDay(for: $0.date) 
            }).count
            
            // 检查是否每天都有记录
            let daysSoFar = calendar.dateComponents([.day], from: startOfWeek, to: now).day ?? 0
            if daysWithActivities >= daysSoFar && daysSoFar >= 7 {
                await checkAndUnlockAchievement(id: "perfect_week")
            }
        } catch {
            print("Error checking perfect week: \(error)")
        }
    }
    
    // MARK: - Helper Methods
    
    private func getHistoricalBestWeek() async throws -> Double {
        // TODO: 实现获取历史最佳周数据
        return 0.0
    }
    
    private func checkAndUnlockAchievement(id: String) async {
        if let index = achievements.firstIndex(where: { $0.id == id }) {
            if !achievements[index].isUnlocked {
                unlockAchievement(at: index)
                await handleAchievementUnlock(achievements[index])
