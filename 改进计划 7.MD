## 问题背景与现状分析

### 当前代码架构

```
ContentView (TabView - 同级页面)
├── Tab 1: DashboardView
│   └── @StateObject achievementService ← 已有主实例 ⚠️
├── Tab 2: CommunityView  
│   ├── @StateObject achievementService ← 重复实例 ❌
│   └── @StateObject rankingService ← 重复实例 ❌
└── Tab 3: SettingsView
```

### 当前问题代码

```swift
// CommunityView.swift - Line 11-12
@StateObject private var achievementService = AchievementService()  // ❌ 新实例
@StateObject private var rankingService = PersonalRankingService()   // ❌ 新实例
```

### 关键发现

| 组件 | 现状 | 结论 |
|------|------|------|
| DashboardView | 已有 @StateObject achievementService | 保持，不移动到 ContentView |
| CommunityView | 重复创建相同服务 | 需要修改为复用 |
| LogActivityView | 已使用依赖注入（init 参数） | 作为参考模式 |

### 原计划问题

| 原计划内容 | 问题 | 修正方案 |
|------------|------|----------|
| 在 ContentView 创建服务 | DashboardView 已有，会冲突 | 使用 @EnvironmentObject 共享 |
| 通过 init 参数传递 | Tab 是同级不是父子 | 使用 .environmentObject() 注入 |

---

## 技术方案（修正版）

### 方案对比

| 方案 | 优点 | 缺点 | 推荐 |
|------|------|------|------|
| **A: EnvironmentObject** | Apple 官方推荐，无需修改服务创建方式 | 需要在 ContentView 设置 | ⭐ 推荐 |
| **B: Singleton** | 简单直接 | 隐藏依赖，难以测试 | ❌ 不推荐 |
| **C: 移动服务到 ContentView** | 集中管理 | 与现有 DashboardView 冲突 | ❌ 不推荐 |

### 修正后的方案 A

```
技术实现:
1. 在 ContentView 使用 @StateObject 创建共享服务
2. 使用 .environmentObject() 注入到 TabView 环境中
3. CommunityView 使用 @EnvironmentObject 接收
4. DashboardView 保持现有 @StateObject（兼容方案）
```

### 修正后的依赖关系图

```
ContentView (TabView + 环境注入)
├── @StateObject achievementService ← 主实例
├── @StateObject rankingService ← 主实例
│
├── DashboardView
│   ├── 保持 @StateObject (现有) ← 继续使用自己的实例
│   └── 也可以使用 @EnvironmentObject (可选)
│
├── CommunityView
│   └── @EnvironmentObject achievementService ← 接收注入
│   └── @EnvironmentObject rankingService ← 接收注入
│
└── SettingsView
    └── (无服务需求)
```

**注意**: DashboardView 继续使用自己的 @StateObject 是安全的，因为：
1. ContentView 的 @EnvironmentObject 会被 DashboardView 的 @StateObject 遮蔽
2. 保持向后兼容，不影响现有功能

## 功能模块设计

### 模块 1: 环境服务容器（ContentView 修改）

**职责**: 在应用根节点创建和注入共享服务到环境

**文件**: ContentView.swift

```swift
struct ContentView: View {
    // 在 ContentView 层级创建服务
    @StateObject private var achievementService = AchievementService()
    @StateObject private var rankingService = PersonalRankingService()
    
    var body: some View {
        TabView {
            DashboardView()
                .environmentObject(achievementService)
                .environmentObject(rankingService)
                .tabItem { ... }
            
            CommunityView()
                .environmentObject(achievementService)
                .environmentObject(rankingService)
                .tabItem { ... }
            
            SettingsView()
                .tabItem { ... }
        }
    }
}
```

**检验标准**:
- [ ] ContentView 创建 achievementService 和 rankingService 实例
- [ ] 所有 Tab 页面都能通过 @EnvironmentObject 访问

### 模块 2: CommunityView 依赖注入

**职责**: 使用 @EnvironmentObject 接收服务而非创建新实例

**文件**: CommunityView.swift

```swift
struct CommunityView: View {
    // 修改：使用 @EnvironmentObject 接收注入的服务
    @EnvironmentObject var achievementService: AchievementService
    @EnvironmentObject var rankingService: PersonalRankingService
    
    // 其他 @State 保持不变
    @State private var totalCO2Saved: Double = 0
    // ...
}
```

**检验标准**:
- [ ] CommunityView 不再创建新的 @StateObject 服务
- [ ] 通过 @EnvironmentObject 访问服务

### 模块 3: 向后兼容处理（可选优化）

**DashboardView 可选优化**: 可以改为使用 @EnvironmentObject，但保持现有代码也能正常工作

```swift
// 可选：保持现有代码兼容
struct DashboardView: View {
    // 优先使用 @EnvironmentObject（如果可用）
    @EnvironmentObject var achievementService: AchievementService?
    
    // 回退到 @StateObject（兼容现有代码）
    @StateObject private var localAchievementService = AchievementService()
    
    // 计算属性：返回可用的服务实例
    private var effectiveService: AchievementService {
        achievementService ?? localAchievementService
    }
}
```

**建议**: 暂不修改 DashboardView，保持现状即可

## 代码实现

### 1. ContentView.swift 修改

```swift
//
//  ContentView.swift
//  suslife
//
//  Main Entry Point - Tab-based Navigation
//  Modified: 添加共享服务实例并注入到环境
//

import SwiftUI

struct ContentView: View {
    @StateObject private var achievementService = AchievementService()
    @StateObject private var rankingService = PersonalRankingService()
    
    var body: some View {
        TabView {
            DashboardView()
                .environmentObject(achievementService)
                .environmentObject(rankingService)
                .tabItem {
                    Label("Home", systemImage: "house.fill")
                }
            
            CommunityView()
                .environmentObject(achievementService)
                .environmentObject(rankingService)
                .tabItem {
                    Label("My Progress", systemImage: "chart.line.uptrend.xyaxis")
                }
            
            SettingsView()
                .tabItem {
                    Label("Settings", systemImage: "gearshape.fill")
                }
        }
        .accentColor(AppColors.primary)
    }
}

#Preview {
    ContentView()
}
```

### 2. CommunityView.swift 修改

```swift
//
//  CommunityView.swift
//  suslife
//
//  Community View - Personal progress tracking
//  Modified: 使用 @EnvironmentObject 接收服务而非 @StateObject
//

import SwiftUI

struct CommunityView: View {
    // 修改：使用 @EnvironmentObject 替代 @StateObject
    @EnvironmentObject var achievementService: AchievementService
    @EnvironmentObject var rankingService: PersonalRankingService
    
    // 直接数据来自 repository
    @State private var totalCO2Saved: Double = 0
    @State private var totalActivities: Int = 0
    @State private var currentStreak: Int = 0
    @State private var isLoading = false
    @State private var errorMessage: String = ""
    @State private var showErrorAlert = false
    @State private var showRetryButton = false
    
    private let repository = CoreDataActivityRepository()
    
    // 删除原有的 init（不需要了）
    // 注意：带有 @EnvironmentObject 的 View 不需要自定义 init
    
    var body: some View {
        ScrollView {
            // ... 现有代码保持不变
        }
        .task {
            await loadData()
        }
    }
}

// Preview 需要手动注入服务
#Preview {
    CommunityView()
        .environmentObject(AchievementService())
        .environmentObject(PersonalRankingService())
}
```
